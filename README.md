## 🚀 Deployment

### Option 1: GitHub Pages (FREE)
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose "main" branch
5. Your site will be live at `https://yourusername.github.io/pr-creations-website`

### Option 2: Netlify (FREE)
1. Connect your GitHub repository to Netlify
2. Netlify will auto-deploy on every push
3. Get a free subdomain or connect custom domain

### Option 3: Vercel (FREE)
1. Import your GitHub repository
2. Vercel will auto-deploy
3. Get a free subdomain or custom domain

## 📝 Customization Guide

### Update Business Information
Edit the following in `index.html`:
- Phone numbers: Search for `+918989418084`
- Email: Search for `prcreation192@gmail.com`
- Address: Search for `Awadhpuri, Bhopal`
- Social media links: Update URLs in footer section

### Add Gallery Images
Replace placeholder gallery cards with your images:
```html
<div class="gallery-card" data-category="birthday">
  <img src="your-image-url.jpg" alt="Birthday Decoration">
  <div class="gallery-overlay">
    <div class="gallery-overlay-label">Your Event Title</div>
  </div>
</div>
```

### Update Testimonials
Replace sample testimonials with real Google reviews:
```html
<div class="testimonial-card">
  <div class="stars">★★★★★</div>
  <p class="testimonial-text">"Your review text here"</p>
  <p class="testimonial-author">- Client Name</p>
</div>
```

### Modify Colors
Edit CSS variables at the top of the `<style>` section:
```css
:root {
  --gold: #c9a84c;
  --rose: #b5446e;
  --cream: #fdf8f0;
  /* ... more colors ... */
}
```

### Update Event Planning Tips
Replace the tip cards with your own tips:
```html
<div class="tip-card">
  <div class="tip-icon">🎨</div>
  <h3>Your Tip Title</h3>
  <p>Your tip description here</p>
</div>
```
